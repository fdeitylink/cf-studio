# kero-edit

A modding tool for Studio Pixel's 2014 platformer [Kero Blaster](https://en.wikipedia.org/wiki/Kero_Blaster).

## Installation

Clone the repo and run `lein run`.

## Usage

    $ java -jar kero-edit-0.1.0-SNAPSHOT-standalone.jar [args]
